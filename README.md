# 🍳 CookSmart AI

**Your Personal AI Cooking Planner**

CookSmart AI is a micro web application built on **Lovable + Gemini AI** that helps users generate a personalized daily cooking plan that is practical, affordable, and easy to execute.  
It acts as a cooking assistant, grocery planner, and budget coach — all in one.

---

## 🚀 Features

- **Personalized Meal Planning**
  - Breakfast, Lunch, Dinner with dish names, ingredients, steps, prep time, and estimated cost.
- **Smart Grocery List**
  - Consolidated shopping list grouped by category (Vegetables, Fruits, Dairy, Meat, Pantry, Spices).
- **Ingredient Substitution Engine**
  - Allergy-friendly, budget-friendly, and cuisine-specific alternatives.
- **Budget Feasibility Dashboard**
  - Real-time cost validation with traffic-light status (✅ Within Budget, ⚠️ Close, ❌ Over).
  - Automatic regeneration with cheaper alternatives if over budget.
- **Daily Cooking To-Do List**
  - Interactive checklist for prep, cooking, and storage tasks.
- **Optimization Tips**
  - Money-saving strategies and food waste reduction suggestions.
- **Cuisine & Geography Selection**
  - Choose cuisines (Thai, Mexican, Indian, Italian, etc.) and adapt plans to local availability and costs.
- **Export Options**
  - Download meal plan and grocery list as PDF or print directly.

---

## 🖥️ Tech Stack

- **Lovable** – No-code platform for rapid prototyping and hosting
- **Gemini AI** – Meal planning, substitutions, budget optimization
- **Responsive UI** – Mobile-first design, dark theme with green accents
- **Hosting** – Auto-hosted via Lovable, deployable to GitHub + Vercel/Netlify
